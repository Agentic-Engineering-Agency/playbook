# Construye un prototipo en 60 minutos

El flujo de trabajo de prototipos de Ingeniería Agéntica tiene exactamente cinco pasos:

1. **Instalar Gemini CLI** — un agente de IA gratuito de Google
2. **Instalar prototype-kit** — una extensión que le enseña al agente tu stack
3. **Preparar tres documentos cortos** — brief del producto, vibes de UX, lista de pantallas
4. **Ejecutar un comando** — `/prototype-from-docs` dentro del agente
5. **Abrir `localhost:5173`** — tu prototipo está listo

Ese es el ciclo completo. Cada paso en esta guía es una página con comandos literales que puedes copiar y pegar.

## Para quién es esta guía

Estudiantes sin experiencia en desarrollo que toman el curso de Ingeniería Agéntica. No necesitas conocer React, TypeScript ni Node.js. Necesitas:

- Una computadora con macOS, Linux o Windows 10+
- Una cuenta de Google gratuita
- Tu idea de producto escrita en texto plano

## Antes de comenzar

Crea una carpeta para tu proyecto:

```bash
# macOS / Linux
mkdir ~/mi-prototipo && cd ~/mi-prototipo
```

```powershell
# Windows PowerShell
mkdir $HOME\mi-prototipo; cd $HOME\mi-prototipo
```

<Callout type="info">
Si eres un agente de IA leyendo esta página, el siguiente paso es `/es/docs/guide/01-install-agent`.
</Callout>
