# Build a prototype in 60 minutes

The Agentic Engineering prototype workflow has exactly five steps:

1. **Install Gemini CLI** — a free AI coding agent from Google
2. **Install prototype-kit** — an extension that teaches the agent your stack
3. **Prepare three short documents** — product brief, UX vibes, screen list
4. **Run one command** — `/prototype-from-docs` inside the agent
5. **Open `localhost:5173`** — your prototype is ready

That is the whole loop. Each step in this guide is a single page with literal, copy-pasteable commands.

## Who this guide is for

Non-developer students taking the Agentic Engineering course. You do not need to know React, TypeScript, or Node.js to follow this guide. You need:

- A computer running macOS, Linux, or Windows 10+
- A free Google account
- Your product idea written down in plain text

## Before you start

Create a folder for your project:

```bash
# macOS / Linux
mkdir ~/my-prototype && cd ~/my-prototype
```

```powershell
# Windows PowerShell
mkdir $HOME\my-prototype; cd $HOME\my-prototype
```
